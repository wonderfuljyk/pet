# 校园萌爪驿站 —— 后端服务

> 校园萌爪驿站后端服务，基于 Spring Boot + MyBatis Plus 构建，提供宠物领养、流浪救助、论坛交流、科普活动等功能的 RESTful API。

## 项目简介

本项目是校园宠物管理系统的后端部分，采用 Spring Boot 框架开发，为前端提供数据接口支持。系统面向普通用户和管理员两类角色，涵盖用户认证、宠物管理、领养审核、论坛互动、数据统计等核心业务模块。

## 技术栈

| 类别 | 技术 |
|------|------|
| 语言 | Java 8 / 11 |
| 框架 | Spring Boot 2.x |
| Web层 | Spring MVC |
| 持久层 | MyBatis Plus |
| 数据库 | MySQL 5.7+ |
| 构建工具 | Maven |
| Excel处理 | Apache POI |
| 工具库 | Hutool |
| 认证方式 | JWT |

| 捐赠信息 | 捐赠记录查看 |
| 文件上传 | 头像、图片等文件上传 |
