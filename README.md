# 校园萌爪驿站 —— 前端界面

> 校园萌爪驿站前端界面，基于 Vue 2.0 + Element-ui 构建，为校园宠物管理系统提供用户端和管理端的可视化操作界面。

## 项目简介

本项目是校园宠物管理系统的前端部分，采用 Vue 2.0 全家桶开发，与后端 Spring Boot 服务通过 RESTful API 进行数据交互。系统面向普通师生（用户端）和管理员（后台管理端），提供宠物领养、流浪救助、论坛交流、科普活动等完整的业务功能。

## 技术栈

| 类别 | 技术 |
|------|------|
| 核心框架 | Vue 2.0 |
| 路由管理 | Vue Router |
| 状态管理 | Vuex |
| UI 组件库 | Element-ui |
| HTTP 请求 | Axios |
| 数据可视化 | ECharts |
| 构建工具 | Vue CLI |

## 主要功能

### 用户端（前台）

| 模块 | 功能说明 |
|------|----------|
| 用户中心 | 注册、登录、找回密码、修改个人信息、修改密码、上传头像 |
| 首页 | 公告列表、流浪宠物推荐、科普文章、热门活动展示 |
| 宠物领养 | 搜索/筛选宠物、查看详情、申请领养、查看我的领养记录 |
| 流浪宠物救助 | 查看需救助的流浪宠物、新增流浪宠物报告 |
| 宠物喂养点 | 查看喂养点环境照片、地址、相关描述 |
| 丢失宠物查看 | 查看走失宠物信息、联系人及联系方式、找回状态 |
| 流浪宠物救助站 | 查看救助站地址、联系人、电话及相关描述 |
| 宠物论坛 | 发布帖子、浏览帖子、关键词搜索、评论互动 |
| 宠物捐赠 | 查看捐赠人及捐赠物资信息 |
| 科普文章 | 阅读宠物科普文章、自动统计阅读量 |
| 热门活动 | 查看宠物相关活动信息 |

## 快速开始

### 环境要求

| 软件 | 版本要求 |
|------|----------|
| Node.js | 14+ |
| npm | 6+ 或 yarn 1.22+ |

### 安装与运行

```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖（首次运行需要）
npm install
# 或使用 yarn
yarn install

# 3. 配置后端接口地址
# 编辑 .env.development 文件，修改以下配置：
VUE_APP_API_BASE_URL=http://localhost:9090

# 4. 启动开发服务器
npm run serve
# 或
yarn serve
